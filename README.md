# overflow-scroll-fade

Add a gradient-based dynamic fade effect to elements with scrollable overflow.

## Usage

```tsx
import { Scroll } from 'overflow-scroll-fade'

const MyScroll = () => (
  <Scroll>
    <p>overflow-scroll-fade</p>
    <p>overflow-scroll-fade</p>
  </Scroll>
)
```
